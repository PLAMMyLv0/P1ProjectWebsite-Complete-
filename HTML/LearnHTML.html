เนื้อหา HTML

<!DOCTYPE html>:

ประกาศประเภทเอกสาร HTML5

<html lang="en">:

เปิดแท็ก HTML และกำหนดภาษาของเอกสาร

<head>:

แท็กที่ใช้สำหรับข้อมูลเมตาและการเชื่อมโยงสไตล์ชีท

<meta charset="UTF-8">:

กำหนดการเข้ารหัสอักขระ

<meta name="viewport" content="width=device-width, initial-scale=1.0">:

ปรับขนาดหน้าจอสำหรับอุปกรณ์พกพา

<title>:

กำหนดชื่อเรื่องที่แสดงในแท็บของเบราว์เซอร์

<link rel="stylesheet" href="styles.css">:

เชื่อมโยงกับไฟล์ CSS ภายนอก

<body>:

เปิดแท็กหลักของเนื้อหาที่แสดงบนหน้าเว็บ

<header>:

ส่วนหัวของเอกสาร

<p> และ <a>:

ตัวอย่างของพารากราฟและลิงก์
