เนื้อหา CSS

CSS เบื้องต้น

body { font-family: Arial, sans-serif; }

กำหนดฟอนต์ของข้อความในเว็บเพจ

h1 { color: blue; }

เปลี่ยนสีข้อความของหัวเรื่อง h1 เป็นสีน้ำเงิน

p { margin: 10px; }

กำหนดระยะห่างรอบข้อความในแท็ก p

.container { width: 80%; margin: auto; }

ตั้งค่าความกว้างและการจัดกลางของ div class "container"

.button { background-color: #4CAF50; color: white; padding: 10px 20px; border: none;border-radius:5px; }

สไตล์ปุ่ม

a { text-decoration: none; color: #007BFF; }

เปลี่ยนสีและลบขีดเส้นใต้ของลิงก์

img { max-width: 100%; height: auto; }
